When working on this site, only edit files in `src/` for website/content changes.

Always follow this process for changes:
1. Pull latest from `master`/`main`.
2. Make a new branch and do development work.
3. Commit, push the new branch, and open a PR.
4. Return the PR link for review.

For commits made in this project, include a footer line at the bottom of the commit message indicating it was assisted with Codex.
